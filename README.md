## Trashcan 개발 개요

### Front

- 반응형 웹페이지로 제작.

> 기본적인 구성

1. 'TrashCan' 제목의 헤더 / 메뉴바 / 내용
2. 메뉴바

   - write : 작성할 수 있도록 함.
   - trashCan : 버리도록 함.

3. 내용 : default, write, read, trash 모드에 따라 바뀜.

<br>

### Back
